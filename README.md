# soso-gameland 🕹️

이름을 넣으면 랜덤으로 결과를 뽑아주는 소소한 웹 게임 모음. 순수 HTML/CSS/JS 한 파일로 동작하며 서버나 빌드 과정이 필요 없습니다.

레트로 오락실 메뉴에서 세 게임 중 하나를 골라 시작합니다.

- **⚔️ 배틀로얄** — 무기를 든 졸라맨들이 링에서 난투를 벌여 마지막 생존자·지정 등수를 가립니다.
- **🪜 사다리타기** — 고전 사다리 + 순간이동 구멍(같은 색·모양끼리 연결). 도착지 미리 공개 토글 지원.
- **🏃 계주 달리기** — 8레인 트랙을 도는 계주. 팀 수(2~8)와 팀당 인원을 자유롭게 조절.

## 실행 방법

`index.html`을 브라우저로 열면 바로 실행됩니다.

```bash
open index.html
```

또는 GitHub Pages로 배포해 링크로 공유할 수 있습니다 (저장소 Settings → Pages → Branch: main).
